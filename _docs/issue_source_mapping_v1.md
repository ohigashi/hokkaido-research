## 補強戦略

1. **報道なし** の ISSUE: 関連する地方紙 ・ 道北ネット ・ 北海道新聞の記事を sources に追加
2. **研究なし** の ISSUE: 北大 / 北見工大 / 酪農学園大 等の研究を sources に追加
3. **個人発信なし** の ISSUE: note 著者 ・ X 等を references に追加 ( ユーザー判断 )
4. **大学のみ厚い** ISSUE: 現場感覚 ( 報道 ・ 個人発信 ) を補強
5. **道庁のみ厚い** ISSUE: 現場・産業の声 ( 業界 ・ 民間 ・ 報道 ) を補強

## 運用ポリシー

- 各 ISSUE は最低 3 種別 ( 公式 + 研究/シンク + 報道 ) のソースを持つことを目標
- 5 種別以上カバーしている ISSUE は「成熟」、3 種別以下は「補強候補」
- 自動ルーチンが新規 ISSUE 追加時に この種別分布を意識するよう prompt を更新
