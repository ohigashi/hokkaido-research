このサイトは、北海道の地域課題を 構造から知り、長期事例から アイデアを得て、日常の 行動変容と課題解決 につなぐことを目指す、地域変革プレイヤー ( 民間・行政・地域おこし ) のための知識基盤です。事実・解釈・取り組み・道外事例・わたしたちにできることまで含め、公開出典付きで整理しています。

「解釈」欄は編集上の見立てであり、出典元の公式見解ではありません。掲載指標 ( 影響地域数・取組件数・道外事例数・構造深度・鮮度 ) は構造データから自動算出した客観値です。取り組み事例は各主体の公開情報にもとづき要約しており、網羅的ではありません。誤りや追加すべき事例があれば更新していく前提のオープンなメモです。
