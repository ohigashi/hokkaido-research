漁業全国 1 位の中身 - 鮭 ・ 秋刀魚 ・ ホタテ ・ カニ ・ タコ

道内漁獲量 965 千 t は全国 1 位 ( 全国の約 25% )。 内訳は鮭 ・ 秋刀魚 ・ ホタテ ・ タコ ・ カニで全国 1 位、 養殖ホタテも国内最大。 海水温上昇 ・ 担い手減少下での持続性課題。

hokkaido-research 編集部

北海道は 海面漁業漁獲量 965 千 t ( 2023 年・全国 1 位 )、 全国の約 25% を占める。 内訳を見ると、 鮭・秋刀魚・ホタテ・タコ・カニで全国 1 位。 養殖ホタテも国内最大規模で、 輸出主力品の 1 つ。

ただし「全国 1 位」の中身は穏やかではない。 鮭は遡上数が長期低下、 秋刀魚は海水温上昇で漁場北上、 ホタテは中国輸出規制で激変、 カニ ・ タコは漁獲量の不安定化。 本稿は 道内漁業の 1 位構造とリスクを整理する。

1. 数値で見る - 道内漁業 1 位品目

全国 1 位品目を整理する。

道内海面漁業漁獲量は 965 千 t ( 2023 年 ・ 全国 1 位 )、 全国の約 25% を占有。 養殖業生産量も国内最大規模。 漁業産出額は 約 3,500 億円 ( 全国 1 位 )。出典: MAFF 漁業生産統計 ・ 帝国書院都道府県別統計 ↗
全国 1 位品目 ( 漁獲量ベース ): 鮭類 ・ 秋刀魚 ・ ホタテ ( 養殖含む ) ・ タコ類 ・ カニ類 ・ 帆立貝 ・ 平目カレイ類等。 これらが道内漁業の柱を形成。出典: 帝国書院・都道府県別統計 ( 北海道 ) ↗
鮭類遡上数は 1990 年代後半 約 8,000 万尾ピーク → 2020 年代前半 約 2,000 万尾 ( 約 1/4 へ縮小 )。 主因は海水温上昇 ・ 餌料変動 ・ 河川環境の複合。出典: 北海道立総合研究機構 ・ 水産研究本部 ↗
秋刀魚漁獲量は 2010 年代 約 25 万 t/年 → 2020 年代前半 約 5 万 t/年 ( 約 1/5 へ縮小 )。 主因は黒潮蛇行 ・ 海水温上昇による漁場北上 ・ 中国 ・ 台湾漁船の公海漁獲増。出典: 水産庁・秋刀魚資源評価 ↗
ホタテ ( 養殖含む ) は道内漁業最大の輸出品の 1 つ。 2023 年中国輸出規制 ( 福島原発処理水関連 ) で大幅減、 米国 ・ ベトナム ・ 台湾 ・ タイ等への販路拡大が進行中。出典: 水産庁・ホタテ貝輸出統計 ↗
仮説: 道内漁業の「全国 1 位」は気候変動 ・ 国際関係 ・ 担い手減少の三重圧力で持続性に課題。 量の維持より、 養殖 ・ 加工 ・ ブランド化 ・ 輸出多角化の組合せが鍵。
推測: 2030 年代、 道内漁業は ( 1 ) 気候適応型品目 ( ブリ ・ サワラ等暖海性魚種 ) の漁獲増、 ( 2 ) 養殖 ・ 陸上養殖の比率拡大、 ( 3 ) 加工 ・ ブランド化による単価向上の方向で再編が進む可能性が高い。

2. 構造的な課題 - 4 つのリスク

リスク現状影響
気候変動海水温上昇 ・ 黒潮蛇行 ・ 漁場北上鮭 ・ 秋刀魚 ・ サンマ漁獲減 ・ ブリ ・ サワラ増
国際関係中国輸出規制 ・ ロシア漁業権 ・ 米中対立ホタテ等輸出主力品の販路リスク
担い手減少道内漁業者 1985 年 約 5 万人 → 2020 年 約 1.5 万人 ・ 60 歳以上 7 割漁業継承困難 ・ 大規模化 ・ 法人化が必要
資源変動海洋環境変化 ・ 餌料変動 ・ 違法漁業資源評価精度 ・ 国際協調が必要
仮説: 4 リスクは連動して進行。 気候変動 → 漁場北上 → 担い手減 → 国際関係悪化 → 輸出リスク → 経営難。 単線対策ではなく、 気候適応 ・ 養殖 ・ 加工 ・ 輸出多角化 ・ 担い手育成を統合した戦略が必要。

3. 中心問い

中心問い: 道内漁業を「全国 1 位の量」で議論するか、 「気候適応 + 養殖 + 加工 + 輸出多角化 + 担い手」の構造で議論するか。 評価軸を「漁獲量」から「付加価値 + 持続性 + 担い手」に組み替えると、 政策設計が変わる。

4. 道外・海外の参考事例

ノルウェー - 養殖鮭で世界 1 位

ノルウェーは養殖鮭で世界 1 位の輸出国。 年間 約 130 万 t ・ 輸出額 約 1 兆円規模。 国家戦略として養殖技術 ・ 環境管理 ・ ブランド化を統合。出典: Norwegian Seafood Council ↗
仮説: ノルウェーモデルは道内鮭養殖の参照点。 但し、 道内は沿岸環境が異なるため陸上養殖 ・ 一部沿岸養殖の組合せで適応する設計が必要。

アイスランド - 漁業 IT + ブランド化

アイスランドは漁業を国家産業として育成、 IT 化 ( 漁獲データ・トレーサビリティ・市場連動 ) + ブランド化で高単価市場を確保。 漁業 GDP 寄与率約 7%。出典: Icelandic Fisheries ↗
推測: アイスランドモデルの IT 化 + ブランド化は道内漁業の付加価値向上の参照点。 道内漁協 ・ 加工業者 ・ 流通の連携で実装可能性が高い。

米国アラスカ - 持続可能漁業 ・ MSC 認証

米国アラスカは厳格な漁業管理 ・ 資源評価 ・ MSC ( 海洋管理協議会 ) 認証で持続可能漁業を確立。 鮭 ・ タラ等の漁獲量を 30 年以上維持。 EU ・ 米国市場で高単価。出典: Alaska Seafood Marketing Institute ↗
仮説: アラスカモデルの本質は「資源管理 + 国際認証 + ブランド化」。 道内も MSC 認証拡大 ・ 道産漁業ブランド統合で参照可能。

5. 取り得る打ち手

短期 ( 1-3 年 )

  • ホタテ輸出販路多角化 ( 米国 ・ ASEAN ・ EU )
  • 陸上養殖 ・ サーモン養殖の実証拡大
  • 漁業者育成 ・ 法人化マッチング
  • 気候変動適応の品種 ・ 漁場戦略

中期 ( 3-10 年 )

  • MSC ・ ASC 等国際認証の道内拡大
  • 漁業 ・ 加工 ・ 流通の道内サプライチェーン統合
  • 気候適応型漁業 ( ブリ ・ サワラ等 ) の制度整備
  • 漁業 IT + データ基盤の整備

長期 ( 10 年以上 )

  • 道内漁業の国際的なブランド確立
  • ノルウェー ・ アイスランド型の養殖大国化
  • 気候変動 ・ 海洋環境のモニタリング体制整備
  • 漁業文化 ・ 担い手の世代継承制度化
