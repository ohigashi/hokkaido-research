海岸侵食への適応 - 道内 4,460km の海岸線をどう守るか

道海岸線は 4,460km で全国の約 14%。 気候変動・海面上昇・台風で侵食が加速。 オランダ Delta Programme・サンドバイパス・グリーンインフラの事例から、 道内 6 振興局別の応用を設計する。

hokkaido-research 編集部
2026-06-13道内一次資料に基づき全面リライト。 道海岸線 4,460km の構造、 海面上昇推計、 オランダ Delta Programme・Sand Engine・Living Shoreline の事例、 道内 6 振興局別打ち手を新規追加。

北海道の海岸線は 約 4,460kmで全国の約 14% を占める。 三方を海に囲まれた地理は道経済の基盤 ( 水産・港湾・観光・物流 ) だが、 気候変動による海面上昇・台風強化・海水温上昇が侵食を加速させている。

本稿はオランダ Delta Programme・サンドバイパス・Living Shoreline といった世界の長期事例から、 道内 6 振興局 ( 道南・道央・道北・オホーツク・十勝・釧路根室 ) 別の応用を整理する。 姉妹記事「気候変動適応」と合わせて読むことで構造が見える。

1. 道内の現状 - 数字で見る海岸侵食

推移で押さえる。 海面上昇と侵食は徐々に進行し、 港湾・漁港・集落・観光地に影響を及ぼし始めている。

北海道の海岸線総延長は約 4,460km で全国 ( 約 35,000km ) の約 14% を占める。 オホーツク・道南・釧路根室の侵食被害が継続報告され、 漁港・観光浜・道路・農地に影響。出典: 国土交通省北海道開発局・海岸防護 ↗
気象庁の海面上昇推計では、 21 世紀末 ( 2076-2095 年 ) に世界平均で +0.43m ( 2℃ シナリオ ) から +0.84m ( 4℃ シナリオ ) と予測。 日本沿岸も同程度の上昇が見込まれ、 道内沿岸線・港湾構造物への影響が長期化。出典: 気象庁・気候変動監視レポート ↗
道海水温は 2023 年 6 月に日本海北部・北海道南東方で過去最高を記録。 海洋生態系の変化と並行して、 沿岸侵食・暴浪・冬期波浪の頻度・強度も変化している。出典: 気象庁・海洋健康診断表 ↗
北海道河川財団・北海道大学・株式会社ドーコンとオランダ HKV・Deltares・デルフト工科大学が連携し、 気候変動適応・海岸防護の研究プロジェクトを進めている。 オランダ Delta Programme の知見を道内に応用する基盤が整いつつある。出典: 北海道河川財団・国際交流 ↗
仮説: 道内海岸侵食は「徐々に進行」する性質で、 単発の災害ではない。 ( 1 ) 海面上昇、 ( 2 ) 暴浪・台風強化、 ( 3 ) 海水温上昇 ( 海洋生態系変化 )、 ( 4 ) 砂浜の人為的縮小 ( 海岸構造物・砂利採取 ) の 4 要因が複合。 長期計画と段階的投資の組合せが必要。
推測: 2030 年に向けて、 道内 6 振興局別に海岸侵食対策計画が再策定される見通し。 オランダ Delta Programme 型の 100 年スパンの計画・段階的投資・自治体住民参加の運営が、 道内沿岸自治体に拡大する可能性が高い。

2. 中心問い

中心問い: 海岸防護を「コンクリート構造物による静的対策」のみで進めるか、 「サンドバイパス・Living Shoreline 等の動的適応」と組合せるか。 評価軸を「現状維持」から「長期適応」にずらすと、 投資判断と空間計画が変わる。

4. 道外・海外の参考事例

オランダ Delta Programme

オランダは 2008 年から Delta Programme で 100 年スパンの気候適応を実装。 2050 年までに堤防 1,400km・水門/ポンプ場 400 強化を計画。 投資総額は年間 約 13 億ユーロ。 自治体・民間・住民の参加型運営。出典: Delta Programme Commissioner ↗
仮説: オランダの本質は「長期計画 + 段階的投資 + 多主体参加」。 道でも同様の 100 年スパン・10 年ごとの見直し・自治体協議の枠組みが必要。

Sand Engine ( オランダ・砂エンジン )

オランダは 2011 年にハーグ近郊の海岸に 2,150 万㎥の砂を一括投入する「Sand Engine」を実装。 自然の波と風で 20 年かけて海岸線全体に砂を分配し、 従来の養浜より低コスト・低環境影響で海岸を維持。出典: Rijkswaterstaat ( オランダ水管理庁 ) ↗
仮説: Sand Engine の革新性は「自然の力を活用」したこと。 個別海岸に養浜を繰り返すより、 1 箇所大型投入で 20 年単位の維持が可能。 道南・釧路の侵食地域で応用余地。

米国 Chesapeake Bay・Living Shoreline

米メリーランド州 Chesapeake Bay は 2008 年から「Living Shoreline」政策を実装。 コンクリート護岸ではなく、 在来植生・牡蠣殻・砂を組合せた自然な海岸線で侵食防止 + 生態系保全 + 漁業振興を両立。 1,500 件超のプロジェクトを実施。出典: Chesapeake Bay Foundation ↗
推測: Living Shoreline 型は道内オホーツク・釧路根室の昆布場・カキ養殖地と相性が良い。 漁業 + 海岸防護 + 観光の三位一体運営が、 2030 年前後の参照モデルになる可能性。

5. 北海道に応用するなら

道内 6 振興局の海岸特性は異なる。 オランダ・米国モデルの組合せで対応。

振興局海岸特性応用すべき打ち手
道南・渡島・檜山観光浜・港湾サンドバイパス + 観光連動
道央・後志ニセコ観光・港湾Sand Engine + 観光
道北・留萌・宗谷漁港・冬期暴浪防波堤強化 + Living Shoreline
オホーツクホタテ養殖場・流氷Living Shoreline + 漁業連動
十勝農地・河口河口堆砂管理 + グリーンインフラ
釧路・根室湿原・水産・カキLiving Shoreline + 国立公園連動
仮説: 道内応用の鍵は「6 振興局単位の海岸計画 + 100 年スパンの段階的投資」。 北海道開発局・道庁・沿岸自治体・大学・住民の 5 主体協議で 10 年ごとに見直す枠組みが必要。
推測: 2030 年前後に道内では「北海道海岸保全長期計画 ( 仮 ) 」が策定される見通し。 オランダ型の 100 年計画・段階的投資・住民参加が、 道内沿岸自治体の標準運営になる可能性が高い。

6. わたしたちにできること

海岸侵食対策は住民・漁業者・観光業者の関わり方で形作られる。

個人として

  • 道内海岸線の状況を継続的に観察 ・ 学ぶ
  • 海岸侵食・気候変動関連のパブコメ参加
  • ふるさと納税で道内海岸保全プロジェクトを選ぶ
  • 海岸でのゴミ拾い・ビーチクリーン参加

企業・組織として

  • 事業の気候リスク評価 ( TCFD・サプライチェーン )
  • Living Shoreline・グリーンインフラ案件への投資・参画
  • 従業員のビーチクリーン・海岸保全 CSR
  • 地域漁業・観光業との海岸保全連携
