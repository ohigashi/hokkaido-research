JR 北海道は維持できるか - 鉄道存続と地域インフラの構造

13 路線で「維持困難」表明、廃線進行中。鉄道存続を「赤字 vs 黒字」ではなく、地域インフラの構造として読み解く。

hokkaido-research 編集部

JR 北海道は 2016年に 「13 線区が維持困難」と表明した。以来、廃線・並行在来線議論・経営支援が続く。

鉄道存続は「赤字 vs 黒字」の議論で語られがちだが、地域インフラの構造として読み解く必要がある。

1. 数値で見る現状

指標数値備考
JR 北海道の年間赤字数百億円規模経常損失・公表
「維持困難」表明線区13 線区 ( 2016 )全営業線の約半数
既に廃線石勝線夕張支線・留萌本線等順次進行
国の経営支援年数百億円規模経営自立支援

「13 線区維持困難」表明から 10年、議論は続いているが、根本的解決は遠い。

2. なぜ JR 北海道は維持困難なのか

  • 人口減・過疎化: 沿線人口の減少で利用者が減る
  • 車社会化: 道内は車所有率が高く、鉄道利用率が低い
  • 広域分散: 距離が長く、維持コストが高い
  • 冬季の追加コスト: 除雪・防雪・設備保守
  • 並行道路の発展: 高速道路・バス網が代替

5 つの構造要因は変えにくい。「JR 北海道が頑張れば黒字化」とはならない構造的問題。

3. 中心問い - 経営自立か地域インフラか

中心問い: 鉄道を「民間企業の経営対象」とするか、「地域インフラ」として位置づけるか。
観点民間企業として地域インフラとして
評価軸黒字・利用者数アクセス保証・地域維持
責任JR 北海道国・道・沿線自治体
費用負担JR 自社公的負担 ( 道路・上下水道と同じ )
参考現行制度欧州型・JR 三島型公的支援強化

道路・上下水道は地域インフラとして公的に整備・維持される。鉄道だけ「民間で黒字化」を求めるのは構造的に無理。

4. 廃線後の地域への影響

  • 高校生・通学者: 進学・通学の選択肢が狭まる
  • 高齢者・車を持たない人: 移動手段の喪失
  • 観光: 鉄道旅行という選択肢の消失
  • 地域経済: 沿線商業・移住希望度の低下
  • 地域アイデンティティ: 「鉄道があった町」の喪失感

廃線の費用 ( 公的負担減 ) は見えるが、廃線の社会的コストは見えにくい。

5. 道外・海外の参考事例

欧州・スイス連邦鉄道

公的補助前提で過疎地の鉄道も維持。「鉄道は公共財」という哲学。利用者数より地域維持を優先。

JR 四国・JR 九州・JR 北海道の三島

国鉄分割民営化時から経営困難。経営支援基金で補填しているが、構造的問題は同じ。

並行在来線の議論

新幹線開業に伴う並行在来線の経営分離。第三セクターでも厳しい運営。北陸・東北の事例から学ぶべき。

6. 取り得る打ち手

短期 ( 1-3年 )

  • 地域インフラとしての公的支援強化
  • 観光列車・イベント列車での需要創出
  • DX・AI 活用での運営コスト削減

中期 ( 3-10年 )

  • 鉄道 + バス + デマンド交通の統合運営
  • 沿線地域との一体運営 ( 上下分離方式 )
  • 観光・教育・物流との連動

長期 ( 10年以上 )

  • 「公共インフラとしての鉄道」の制度的定着
  • 次世代鉄道技術 ( 自動運転・新エネルギー ) の導入
  • 地域住民の鉄道との関わり方の再構築

7. わたしたちにできること

JR 北海道存続は「他人事」ではない。日常の利用・関心・議論への参加で、地域インフラの未来を支えられる。

個人として

  • 出張・旅行で JR 北海道を意識的に利用する
  • 観光列車・イベント列車を活用する
  • 鉄道存続議論への関心・意見表明
  • 地域メディアで鉄道の現状を発信

企業・組織として

  • 業務利用での鉄道優先の社内ルール
  • JR 北海道との CSR・物流連携
  • 観光列車・イベント企画への協力
まとめ: JR 北海道存続は経営問題ではなく、地域インフラの位置づけの問題。鉄道を道路・上下水道と同じ公共財として捉え直す制度改革と、わたしたちの利用・関心が、長期的な維持を支える。
