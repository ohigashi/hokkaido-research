尾道市 - 空き家再生で「坂のまち」を蘇らせる 15 年

道内空き家 60 万戸 ( 空き家率 17% ) ・ 地方中心市街地衰退の課題に対し、 広島県尾道市は NPO 空き家再生プロジェクトで「坂のまち」を 15 年で蘇らせた。 道内地方都市 ・ 観光地への参照点。

hokkaido-research 編集部

道内空き家は約 60 万戸 ・ 空き家率 17% ( 全国 13.8% )。 地方の地価下落と並走、 中心市街地の衰退も道内中核市で課題。 「壊す」だけでなく「活かす」道筋が必要。

広島県尾道市 ( 人口 約 13 万人 ) は、 2007 年から NPO 法人 尾道空き家再生プロジェクトが「坂のまち」の空き家を再生 ・ 活用 ・ 移住促進してきた。 古民家カフェ ・ ゲストハウス ・ 工房等が集積、 関係人口 ・ 移住者を生み出す道内地方都市 ・ 観光地への参照点。

1. 道内の課題と数値

道内空き家数は 2023 年住宅 ・ 土地統計で約 60 万戸 ・ 空き家率 17% ( 全国 13.8% )。 地方ほど高く、 道北 ・ 道東 ・ 道南で 20% 超の自治体も。出典: 総務省・住宅 ・ 土地統計調査 ↗
道内地価二極化: 札幌 ・ ニセコ ・ 千歳で上昇、 道北 ・ 道東 ・ 道南で -2 ~ -5% 下落。 中心市街地衰退と空き家増加が連動。出典: 国土交通省・地価公示 ・ 北海道庁 ↗
仮説: 空き家を「負債」ではなく「地域資源」と見る転換が必要。 尾道モデルは道内地方都市 ・ 観光地の参照点。

2. 尾道市の数値

尾道市人口は約 13 万人 ・ 高齢化率 35% 超。 中心市街地の空き家率は一時 30% 超。 2007 年から NPO 法人 尾道空き家再生プロジェクトが活動開始。出典: 尾道空き家再生プロジェクト ↗
2024 年時点で約 250 軒の空き家を再生 ・ 活用 ・ 移住者マッチング。 古民家カフェ ・ ゲストハウス ・ 工房等が集積、 観光客 ・ 関係人口も増加。出典: 尾道空き家再生プロジェクト・実績報告 ↗
尾道市は 2020 年「日本遺産」認定 ( 坂のまち 文脈 )。 観光 ・ 移住 ・ 空き家再生が統合的に進行。出典: 尾道市・公式サイト ↗

4. 道内への翻訳可能性

  • 観光地 ( 函館 ・ 小樽 ・ 室蘭 ) — 歴史建築 + 空き家活用
  • 炭鉱跡 ( 夕張 ・ 美唄 ・ 三笠 ) — 産業遺産 + 空き家
  • 中山間 ( 江差 ・ 上ノ国 ・ 松前 ) — 歴史町並み + 移住
  • 中核市 ( 旭川 ・ 帯広 ・ 釧路 ) — 中心市街地再生
推測: 道内では「冬季の建物維持コスト」「断熱性能」が空き家活用の制約。 但し、 古民家ブームと観光 ・ 移住需要の追い風で参照可能性は高まる見通し。

5. 限界と論点

論点: ( 1 ) NPO 主導のスケーラビリティ、 ( 2 ) 建築士 ・ 大工の世代継承、 ( 3 ) 文脈づくりの長期コスト、 ( 4 ) 観光地化のオーバーツーリズム副作用。
