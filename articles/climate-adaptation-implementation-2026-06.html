道平均気温 +3℃ 時代の適応 - 北海道が直面する変化

2023 年夏は平年比 +3.0℃、 2024 年は +2.2℃ で記録更新。 札幌 36.1℃ ・ 海水温過去最高。 21 世紀末には +3.45℃ シナリオも。 オランダ・デンマーク・京都の適応事例から、 道内で先回りすべき設計を読み解く。

hokkaido-research 編集部
2026-06-13道内一次資料に基づき全面リライト。 2023-2024 年の気温・海水温推移、 道気候変動適応センター、 オランダ Delta Programme・デンマーク豪雨計画・京都市熱中症対策の事例、 道内 4 分野の打ち手を新規追加。

北海道の気候は急激に変わっている。 2023 年夏は平年比 +3.0℃ ( 観測史上 1 位 ) 、 2024 年夏は +2.2℃ ( 2 位 ) 。 2023 年 8 月には札幌で 36.1℃ を記録し、 同月の日本海北部・道南東海域の海面水温は 6 月として過去最高に達した。

気候変動はもはや「将来予測」ではなく 進行中の現実だ。 本稿は道内一次資料で気温・海水温・降水量の推移を把握し、 オランダ Delta Programme・デンマーク豪雨計画・京都熱中症対策の事例から、 道内 4 分野 ( 農業・水産・防災・健康 ) での適応を設計する。

1. 道内の現状 - 数字で見る気候変動

推移で押さえる。 気温・海水温・極端降水の 3 軸で、 道内が直面する変化を確認する。

北海道の夏季 ( 6-8 月 ) 平均気温は 2023 年に平年比 +3.0℃ ( 観測史上 1 位 ) 、 2024 年は +2.2℃ ( 2 位 ) 。 2 年連続で過去最高水準を記録し、 長期トレンドでも上昇が継続。出典: 札幌管区気象台・北海道地方の気候変動 ↗
札幌の年最高気温は 2023 年 8 月 23 日に 36.1℃ ( 観測史上最高 ) 。 1971-2000 年平均 ( 約 32℃ ) を 4℃ 以上上回り、 熱中症搬送者数も急増。出典: 気象庁・札幌の観測データ ↗
2023 年 6 月の日本海北部・北海道南東方の海面水温は同月として過去最高 ( 統計開始以降 ) 。 サケ・サンマの分布変化・ブリ北上の主要因に。出典: 気象庁・海洋の健康診断表 ↗
21 世紀末 ( 2076-2095 年 ) の道周辺海面水温上昇は、 4℃ シナリオで +3.45℃ ( ±1.30 ) 、 2℃ シナリオでも +1.13℃ ( ±0.65 ) と推計。 海洋生態系・水産業への影響が継続。出典: 気象庁・気候変動監視レポート ↗
道は 2021 年 4 月に「北海道気候変動適応センター」を設立。 札幌管区気象台・道立総合研究機構・大学等と連携し、 農業・水産・防災・健康の 4 分野で適応研究を進めている。出典: 北海道気候変動適応センター ↗
仮説: 道内気候変動の影響は「個別現象」ではなく「複合的な構造変化」。 ( 1 ) 農作物 ( 米・畑作・果樹 ) の品種・栽培時期の変化、 ( 2 ) 水産業 ( サケ減・ブリ北上 ) 、 ( 3 ) 短時間豪雨・洪水・土砂災害、 ( 4 ) 熱中症・新規感染症の拡大、 の 4 つが同時並行で進む。
推測: 2030 年に向けて、 道内では「品種転換 + 養殖シフト + 都市豪雨対策 + 熱中症対策」の 4 領域同時並行の適応が本格化する見通し。 道庁の気候変動適応センター主導で、 大学・民間・自治体の連携が拡大する可能性が高い。

2. 中心問い

中心問い: 気候変動を「外部要因」として対策するか、 「道内産業・生活の構造転換のきっかけ」として活用するか。 評価軸を「現状維持のコスト」から「構造転換の機会」へずらすと、 投資判断が変わる。

気候変動適応に投じた予算・施設・知見のうち、 何が地域に残るかを 4 種に整理。

資産種別中身残る条件失敗パターン

4. 道外・海外の参考事例

オランダ Delta Programme ( 国家適応計画 )

オランダは 2008 年から Delta Programme で 100 年スパンの気候適応を実装。 ( 1 ) 水管理・洪水対策、 ( 2 ) 淡水供給、 ( 3 ) 空間計画の 3 分野で年間 約 13 億ユーロを投資。 2050 年・2100 年シナリオを公式に組み込み、 自治体・民間・住民の参加型運営。出典: Delta Programme Commissioner ↗
仮説: オランダモデルの本質は「長期計画 + 自治体参加 + 段階的投資」の 3 点セット。 道でも気候変動適応センターを核に、 100 年スパンの計画と自治体・住民参加が必要。

デンマーク・コペンハーゲン豪雨計画

コペンハーゲンは 2011 年大豪雨 ( 損害約 10 億ユーロ ) を機に、 2012 年から「Cloudburst Management Plan」を実装。 約 300 のグリーン・ブルー・グレー インフラを 20 年計画で配置し、 投資総額 約 10 億ユーロ。 都市公園・道路・地下貯留を統合運営。出典: City of Copenhagen ↗
仮説: コペンハーゲンの鍵は「災害コスト → 適応投資」の財源化。 札幌・函館・旭川等の道内中核市でも、 短時間豪雨対策にグリーンインフラを組合せた長期計画が現実解。

京都市・熱中症対策と都市冷却

京都市は 2018 年から「熱中症予防情報 + クールスポット + 都市緑化」を統合運営。 2022 年に「クールシェア」拠点を市内 800 箇所に拡大し、 熱中症搬送者数を 5 年で約 20% 削減。出典: 京都市・熱中症対策 ↗
推測: 道内では札幌・旭川等で 2030 年に向けて同様のクールシェア + 都市緑化が本格導入される見通し。 高齢者・子ども・観光客への熱中症対策が、 公衆衛生の新軸になる。

5. 北海道に応用するなら

道内 4 分野 ( 農業・水産・防災・健康 ) で、 オランダ・デンマーク・京都モデルを組合せる。

分野現状の課題応用すべき打ち手
農業米・畑作の作柄不安定化、 果樹適地北上品種転換 + 栽培時期調整 + 気候保険
水産サケ減・ブリ北上・養殖適地変化魚種ポートフォリオ + 陸上養殖 + ITQ
防災短時間豪雨・洪水・土砂災害コペンハーゲン型グリーンインフラ + 警報統合
健康熱中症・新規感染症・高齢者リスク京都型クールシェア + 都市緑化 + 警報
仮説: 道内応用の鍵は「4 分野の同時並行進行」。 オランダの 100 年計画・自治体参加・段階的投資のフレームを、 道気候変動適応センター・大学・自治体・民間で運営する。 1 分野だけ突出しても全体が崩れる構造。
推測: 2030 年前後に道庁・国の連携で「北海道気候変動適応計画 ( 仮 ) 」が策定され、 100 年スパンの段階的投資・自治体参加・住民教育が本格化する可能性が高い。

6. わたしたちにできること

気候変動適応は政策のみで進まない。 個人・企業の日常選択が、 規範を形作る。

個人として

  • 気候変動・適応の最新情報を継続的に学ぶ
  • 熱中症・豪雨・土砂災害への日常的な備え
  • ふるさと納税で道内適応事業・養殖・農業転換を選ぶ
  • 日常生活で省エネ・脱炭素の選択を継続

企業・組織として

  • 事業の気候リスク評価 ( TCFD・サプライチェーン )
  • 気候変動適応・脱炭素の R&D 投資
  • 従業員の熱中症対策・在宅勤務制度
  • 地域適応プロジェクト・防災への協賛
