上勝町 - 葉っぱビジネス + ゼロウェイストの 40 年戦略

道内一次産業の高齢化 ・ ゴミ処理コスト ・ カーボンニュートラルの課題に対し、 徳島県上勝町は 1980 年代から「いろどり」葉っぱビジネス + 2003 年ゼロウェイスト宣言の組合せで「お年寄り × 環境」を両立。

hokkaido-research 編集部

道内農業の高齢化率 70% は深刻課題。 60 歳以上比率は全国平均を上回り、 経営継承困難な経営体が連鎖発生する見通し。 一方で道内ゴミ処理 ・ カーボンニュートラル目標も同時進行課題。

徳島県上勝町 ( 人口 約 1,400 人 ) は、 1980 年代から 「いろどり」葉っぱビジネスでお年寄りに月収 100 万円超の事業者を生み出し、 2003 年に 全国初のゼロウェイスト宣言でゴミリサイクル率 80% 超を実現。 「お年寄り × 環境」の同時最適化モデル。

1. 道内の課題と数値

道内農業従事者の 60 歳以上比率は約 70%、 50 歳未満は 15% 未満。 経営継承困難な経営体が連鎖発生する見通し。出典: MAFF 農林業センサス ↗
道庁 ゼロカーボン北海道は 2030 年に 2013 年度比 48% 削減目標。 道内ゴミ処理 ・ リサイクル率向上は地方自治体の課題。出典: 北海道庁・ゼロカーボン北海道 ↗
仮説: 上勝モデルは「高齢者活用 + 環境」の二重ボトムライン。 道内中山間 ・ 過疎地域の参照点。

2. 上勝町の数値

上勝町人口は 約 1,400 人 ( 2024 年 )、 高齢化率 50% 超。 「いろどり」事業者は約 200 人、 平均年齢 70 歳超、 売上ピークで年商 2.5 億円。 70 歳超でスマホ ・ タブレット運用。出典: 株式会社いろどり ↗
2003 年「ゼロウェイスト宣言」、 ゴミ 45 分別 ・ 焼却ゼロ目標。 2024 年時点のリサイクル率 80% 超 ( 全国平均 約 20% )。 ゼロウェイストセンター「WHY」が拠点。出典: 上勝町・ゼロウェイスト推進 ↗
2020 年「Zero Waste Town Kamikatsu」が国際的環境表彰を多数受賞。 視察 ・ 観光客 ・ 移住者が継続的に流入。出典: 上勝町・公式サイト ↗

4. 道内への翻訳可能性

  • 中山間自治体 ( 美深 ・ 中川 ・ 滝上 ) — いろどり型小規模ビジネス
  • 脱炭素先行地域 ( 上士幌 ・ 厚真 ) — ゼロウェイスト統合
  • 酪農地域 ( 別海 ・ 標茶 ) — 廃棄物 ・ バイオマス連動
  • 観光地 ( ニセコ ・ 富良野 ) — ゼロウェイスト観光ブランド
推測: 道内では「冬季の収集インフラ」「広域分散」が制約。 上勝の細密 45 分別をそのまま適用は難しいが、 「高齢者 + IT + 地域資源」の組合せは参照可能。

5. 限界と論点

論点: ( 1 ) いろどりは料亭 ・ 高級飲食店需要に依存、 ( 2 ) 高齢者の体力限界、 ( 3 ) ゼロウェイストの実装コスト、 ( 4 ) 視察 ・ 観光依存リスク。
