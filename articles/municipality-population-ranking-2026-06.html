道内 179 市町村別人口動態ランキング - 増える 6 ・ 減る 173

2020-2025 年で道内人口増加した自治体は 6 のみ。 残り 173 市町村は減少。 増加組 ( 赤井川 ・ 南幌 ・ 東川 ・ ニセコ ・ 倶知安 ・ 恵庭 ) の共通要因と、 減少 173 自治体の類型化。

hokkaido-research 編集部

2025 年国勢調査で道内人口は 戦後初の 500 万人割れ。 5 年で 4.2% 減 ( 全国 -2.2% )。 だが、 道内 179 市町村で増加したのはわずか 6 自治体。 残り 173 自治体は減少した。

増加組 ( 赤井川 ・ 南幌 ・ 東川 ・ ニセコ ・ 倶知安 ・ 恵庭 ) の共通要因と、 減少 173 自治体の類型化を住基ネット ・ 国勢調査データで整理する。

1. 数値で見る - 道内 179 市町村の動態

推移で押さえる。 全道平均と市町村別の二重構造。

2025 年国勢調査で道内人口は約 498 万人 ( 戦後初の 500 万人割れ )。 2020 → 2025 で -4.2% ( 全国 -2.2% )。 道内人口の約 4 割が札幌市に集中、 札幌一極集中構造。出典: 総務省統計局・国勢調査 ↗
2020-2025 で人口増加した道内自治体は 6 のみ: 赤井川村 ・ 南幌町 ・ 東川町 ・ ニセコ町 ・ 倶知安町 ・ 恵庭市。 残り 173 市町村は減少。 過疎指定 152 市町村 ( 道内 85% ) は減少率がさらに大きい。出典: 北海道庁・人口ビジョン 2024 年改訂版 ↗
道内 179 市町村のうち、 2070 年に人口が現在の 50% を下回ると推計される自治体は約 90 ( 約半数 )。 道南 ・ 道北の小規模自治体が中心。出典: 国立社会保障・人口問題研究所・地域別将来人口推計 ↗
住基ネット月次データ ( 総務省 ) では、 道内 179 市町村別の自然動態 ( 出生 - 死亡 ) ・ 社会動態 ( 転入 - 転出 ) を月次で取得可能。 増加 6 自治体は社会動態 ( 転入超過 ) で増加、 減少自治体は両方の負が連動。出典: 総務省・住民基本台帳人口・人口動態 ↗
仮説: 道内人口減は「全道平均」では捉えきれない。 増加 6 自治体は明確なターゲット ・ 文脈 ( 国際リゾート ・ 写真文化 ・ 子育て層 ・ ニセコ広域 ) を持ち、 減少 173 は構造的に類型化できる。
推測: 2030 年代の道内人口減は加速する見通し。 但し、 「増加組モデル ( ターゲット明確化 + 文脈づくり + 関係人口 )」を採用する自治体は微増 ・ 維持に転じる可能性がある。 道庁 ・ 市町村 ・ 民間の連携が鍵。

2. 増加 6 自治体の共通構造

増加組を解剖すると 4 つの共通要因が見える。

自治体増加率 ( 5 年 )ターゲット ・ 文脈
赤井川村 ( 後志 )+ ( 微増 )ニセコ広域 ・ リゾート波及
南幌町 ( 空知 )+ ( 微増 )札幌圏子育て層 ・ 住宅地
東川町 ( 上川 )+ ( 維持 )写真の町 ・ 文化教育 ・ 国際留学
ニセコ町 ( 後志 )+ ( 増加 )国際リゾート ・ インバウンド
倶知安町 ( 後志 )+ ( 増加 )国際リゾート ・ ニセコ広域
恵庭市 ( 石狩 )+ ( 5 年連続 )札幌圏 ・ 子育て世代 ・ 学生
仮説: 増加 6 自治体の共通要因は ( 1 ) ターゲット明確化 ( 「誰に選ばれるか」)、 ( 2 ) 長期文脈づくり ( 東川写真 40 年 ・ ニセコ国際化 30 年 )、 ( 3 ) 札幌圏 / 国際リゾート波及、 ( 4 ) 子育て ・ 教育 ・ 文化等の付加価値。 全方位の移住促進ではない。

3. 中心問い

中心問い: 道内 179 市町村に「全方位移住促進」を続けるか、 「増加 6 自治体モデル」 + 「減少 173 のソフトランディング」を分けて設計するか。 評価軸を「人口維持」から「ターゲット適合 + 関係人口 + 残る資産」に組み替えると、 政策設計が変わる。

4. 道外・海外の参考事例

島根県海士町 - 「ないものはない」

島根県海士町は離島で過疎指定、 隠岐島前高校の魅力化 ・ 島留学 ・ 地域商社で I/U ターン促進。 人口減少局面でも持続的な町づくりを 20 年継続。出典: 海士町公式サイト ↗
仮説: 海士町モデルの本質は「教育を軸にしたターゲット絞り込み」。 道内の東川町写真モデルと共通構造。 減少自治体でも参照可能。

徳島県神山町 - サテライトオフィス + 教育移住

徳島県神山町はサテライトオフィス誘致 + 神山つなぐ公社 + 神山まるごと高専で社会増を実現。 人口減少局面でも挑戦的な町づくりを継続。出典: イン神山・NPO グリーンバレー ↗
推測: 神山モデルの「サテライトオフィス + 教育」は道内でリモート対応自治体 ( 札幌圏近郊 ・ 観光地 ) で参照可能。 但し、 道内の物理的距離 ・ 寒冷条件への適応が必要。

ドイツ ・ ライプチヒ - 「縮小都市の魅せ方」

旧東ドイツ ・ ライプチヒは 1990 年代に人口 50 万 → 40 万に減少後、 文化 ・ 創造産業 ・ 都市再生で 2010 年代以降に再増加。 「縮小都市の戦略的回復」モデル。出典: City of Leipzig ↗
仮説: ライプチヒモデルは札幌 ・ 旭川等の道内中核都市の参照点。 縮小局面での再投資 ・ 創造産業育成 ・ 都市再生の組合せ。
