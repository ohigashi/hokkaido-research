道内有効求人倍率トラッカー - 介護・看護・建設で 3 倍超

道内有効求人倍率は全体 1.0-1.2 台 ( 全国 1.2 前後 ) だが、 職種別で 3 倍超 ( 介護 ・ 看護 ・ 建設 ・ 運輸 ) と 0.3 ( 専門事務 ) の格差。 札幌 ・ 道東 ・ 道北で地域差も大。

hokkaido-research 編集部

道内有効求人倍率は全体で約 1.0-1.2 台。 全国 ( 1.2 前後 ) とほぼ同水準に見える。 だが、 職種別 ・ 地域別に分解すると、 「単一指標で語れない」労働市場の二極化が浮かぶ。

介護 ・ 看護 ・ 建設 ・ 運輸 ・ 警備で 3 倍超。 一方で専門事務 ・ 一般事務は 0.3-0.5。 同じ「道内」で 10 倍以上の需給格差が並存。 札幌 ・ 道東 ・ 道北の地域差も大きい。 本稿は道内労働市場の構造を整理する。

1. 数値で見る - 道内有効求人倍率

推移と内訳で押さえる。

道内有効求人倍率 ( 常用 ) は 2024 年平均 1.10、 2025 年平均 1.15 前後で推移。 全国平均 1.20 前後とほぼ同水準。 但し、 道内は地域差 ・ 職種差が全国比で大きい。出典: 北海道労働局・労働市場年報 ↗
職種別道内有効求人倍率 ( 2024 年平均 ) : 介護 ・ 福祉 約 3.5、 看護 ・ 医療 約 3.0、 建設 ・ 土木 約 4.0、 運輸 ・ 物流 約 2.5、 警備 ・ 保安 約 5.0、 一方で一般事務 約 0.4、 専門事務 約 0.5。 10 倍超の需給格差。出典: 北海道労働局・職種別求人求職状況 ↗
地域別道内有効求人倍率 ( 2024 年平均 ) : 札幌 約 0.95 ( 全道で最低 )、 道東 ( 釧路 ・ 帯広 ・ 北見 ) 1.1-1.3、 道北 ( 旭川 ・ 稚内 ) 1.2-1.4、 日胆 ( 苫小牧 ・ 室蘭 ) 1.3-1.5。 札幌に求職者が集中、 地方で人手不足。出典: 北海道労働局・安定所別月間有効求人倍率 ↗
道内製造業従事者は 1985 年 約 30 万人 → 2024 年 約 16 万人 ( 約 半減 )。 一方、 サービス業従事者は同期間で約 1.5 倍に増加。 産業構造の転換が労働市場を二極化。出典: 北海道庁・労働力調査 ↗
仮説: 「道内全体 1.1 倍」という単一指標は労働市場の実態を隠す。 職種 ・ 地域別に分解すると、 介護 ・ 建設 ・ 運輸等の人手不足、 札幌の求職者集中、 専門事務の需給逆転、 道東 ・ 道北の人手不足という二極化が見える。
推測: 2030 年代、 道内労働市場は ( 1 ) 介護 ・ 建設 ・ 運輸の更なる人手不足、 ( 2 ) 札幌への集中緩和を狙うリモート・二地域勤務、 ( 3 ) 外国人技能実習 → 特定技能への移行、 ( 4 ) スキル不適合解消のリスキリングの 4 軸で動く見通し。

3. 中心問い

中心問い: 道内労働市場を「全体 1.1 倍」で議論するか、 「職種 ・ 地域 ・ スキルの構造」で議論するか。 評価軸を「平均値」から「職種別 + 地域別 + スキル別」に組み替えると、 雇用政策 ・ リスキリング ・ 移住政策が変わる。

4. 道外・海外の参考事例

ドイツ ・ デュアルシステム ( 職業教育 + 企業実習 )

ドイツは中学卒業後の若者の約半数がデュアルシステム ( 職業学校 + 企業実習 ) に進学。 介護 ・ 建設 ・ 製造 ・ IT 等で 350 職種を制度化。 若年層失業率 6% で EU 最低水準。出典: Federal Institute for Vocational Education and Training ↗
仮説: ドイツモデルは道内の介護 ・ 建設 ・ 運輸等の人手不足職種の人材育成の参照点。 道内大学 + 高校 + 職業訓練校 + 企業の連携で再現可能。

シンガポール ・ SkillsFuture

シンガポールは 2015 年から SkillsFuture プログラムで国民全員に 500 ドルの学習クレジットを付与、 リスキリングを国家戦略化。 介護 ・ IT ・ 製造業の需給ギャップを縮小。出典: SkillsFuture Singapore ↗
推測: SkillsFuture モデルは道内 ( 道庁 + 経産省 ) でリスキリング制度の参照点。 一般事務 ・ 専門事務の求職者を介護 ・ 建設 ・ 運輸へ移行する仕組み。

豪州 ・ 地方雇用優遇移住制度

豪州は 2019 年から地方雇用優遇移住制度 ( Regional Visa ) を導入。 地方で 3 年勤務すれば永住権申請が可能。 介護 ・ 建設 ・ 農業等の人手不足解消。出典: Australian Government Department of Home Affairs ↗
仮説: 豪州モデルは道内の道東 ・ 道北 ・ 離島の人手不足解消への参照点。 外国人特定技能 + 永住権の道筋設計で道内地方への定着促進。

5. 取り得る打ち手

短期 ( 1-3 年 )

  • 道内有効求人倍率の職種 ・ 地域別可視化ダッシュボード
  • リスキリング制度の道内全域展開
  • 外国人特定技能の受入れ拡大
  • 札幌から道東 ・ 道北への移住支援強化

中期 ( 3-10 年 )

  • ドイツ型デュアルシステムの道内導入
  • シンガポール SkillsFuture 型リスキリング制度
  • 豪州型地方雇用優遇移住の道内版
  • リモート ・ 二地域勤務の制度整備

長期 ( 10 年以上 )

  • 道内労働市場の職種 ・ 地域別最適配置
  • 多文化共生 ・ 外国人定住の制度化
  • AI ・ ロボティクスによる介護 ・ 建設の省人化
  • 労働 ・ 教育 ・ 移住の統合政策
