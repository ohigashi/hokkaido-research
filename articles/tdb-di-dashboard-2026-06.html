道内 DI ダッシュボード - 業種・地域で読む景気の今

TDB 月次景気 DI は道内経済の温度計。 2026 年 5 月時点で総合 DI 40.5・道東 38.5・日胆 42.9・農林水産 +5.6pt・不動産 -2.6pt。 業種・地域別の動きを読む入口。

hokkaido-research 編集部

「北海道経済はどうなっているか」を 1 つの数字で語ろうとすると道民経済計算や GDP が出てくる。 だが、 月次で動く景気感を取りたいなら帝国データバンク ( TDB ) の 景気 DI ( Diffusion Index )が有用だ。

2026 年 5 月時点の道内総合 DI は 40.5 ( 前月比 +0.4pt ・ 3 ヶ月ぶり改善 )。 全国 41.6 をやや下回るが、 業種別・地域別の動きを切り出すと景気の構造変化が見える。 本稿は DI を読む入口を整理する。

1. 数値で見る - 2026 年 5 月時点の DI

TDB DI は道内 2,000 社程度を対象にした景気感調査。 50 を境に上下で景気感を判定。

道内総合 DI は 2026 年 5 月 40.5 ( 前月比 +0.4pt ・ 3 ヶ月ぶり改善 )。 全国は 41.6。 先行き DI ( 3 ヶ月後 ) は 41.3 で緩やかな改善見通し。出典: TDB 北海道経済レポート 2026 年 5 月版 ↗
地域別 DI: 道東 38.5 ( +2.6pt ・ 3 ヶ月ぶり改善 )、 日胆 ( 日高・胆振 ) 42.9 ( +1.7pt ・ 2 ヶ月連続改善 )。 道東は半導体千歳の余波より一次産業・観光が主体で、 日胆は港湾 ・ 製造業の影響が大きい。出典: TDB 北海道経済レポート ↗
規模別 DI: 大企業 40.8 ( -1.6pt ・ 3 ヶ月連続悪化 )、 中小企業 40.4 ( +0.8pt )、 小規模企業 40.1 ( +0.3pt )。 大企業が悪化基調なのは円安 ・ 中東情勢 ・ 中国経済鈍化等の国際要因が反映。出典: TDB 北海道経済レポート ↗
業種別 DI: 改善 = 農林水産 42.6 ( +5.6pt ・ 4 ヶ月ぶり )、 運輸・倉庫 36.7 ( +3.4pt ・ 5 ヶ月ぶり )。 悪化 = 不動産 42.4 ( -2.6pt ・ 2 ヶ月連続 )。 不動産は利上げ ・ 資材コスト ・ 新規物件取得ハードル上昇が反映。出典: TDB 北海道経済レポート ↗
仮説: 道内景気は「総合 DI 40 前後」で頭打ち感がある。 但し、 千歳ラピダス進出に伴う土木建築需要、 道東農林水産の改善、 道央製造業の動きが地域 ・ 業種別に景気構造を分化させつつある。
推測: 2026 年後半 - 2027 年は半導体千歳本格稼働 ( 2027 年 ) + 道東農林水産改善 + 観光回復で道内 DI が 45 前後に回復する見通し。 但し、 円安・中東情勢・米中対立等の国際要因が引き続き不確実性をもたらす。

2. 業種別 ・ 地域別の構造

DI を分解すると、 道内経済の現状が立体的に見える。

業種5 月 DI前月差背景
農林水産42.6+5.6pt輸出回復 ・ 道産品需要 ・ 春期繁忙
運輸・倉庫36.7+3.4pt物流回復 ・ 半導体千歳関連需要
製造( 道平均圏 )改善基調千歳ラピダス進出効果
小売・サービス( 道平均圏 )横ばい観光回復は道央 ・ 道南限定
不動産42.4-2.6pt利上げ ・ 資材コスト ・ 新規取得難
建設( 道平均圏 )改善基調千歳半導体関連土木建築
仮説: 道内経済は「千歳半導体 + 道東一次産業」が牽引、 「不動産 + 観光 ( 道北 ・ 道東 )」が下押し。 単一の「道内景気」ではなく、 業種 ・ 地域別の景気構造として読むべき段階。

3. 中心問い

中心問い: 道内景気を「総合 DI」で議論するか、 「業種別 + 地域別 + 規模別」の構造で議論するか。 評価軸を「平均値」から「分散 + 推移 + 業種別」に組み替えると、 景気対策 ・ 産業政策設計が変わる。

4. 道外・海外の参考事例

日銀短観 - 全国の景気観測指標

日本銀行が四半期発表する短観 ( 全国企業短期経済観測調査 ) は全国 1 万社以上を対象にした景気判断 DI。 業種・規模別の細分化、 国際比較も可能。 道内 DI 補完情報として有用。出典: 日本銀行・短観 ↗
推測: TDB DI ( 月次・道内 ) と日銀短観 ( 四半期・全国 ) を統合すると、 道内独自の景気構造変化と全国比較の両方が見える。

ifo Business Climate Index ( ドイツ )

ドイツの ifo 経済研究所が発表する月次景気指数 ( ifo Business Climate Index ) は EU 全体の景気観測指標として広く使われる。 業種・地域別の細分化 ・ 30 年以上の連続データ。出典: ifo Institute ↗
仮説: ifo モデルは道内 DI の精度向上の参照点。 業種・地域別の細分化 ・ 連続性 ・ 国際比較を強化すると、 政策インプットとしての価値が高まる。

5. 取り得る打ち手

短期 ( 1-3 年 )

  • DI 月次データの道内向け可視化ダッシュボード整備
  • 業種・地域別景気レポートの定期発信
  • DI を意思決定に組み込む中小企業向けガイド
  • 千歳ラピダス進出効果の継続的なモニタリング

中期 ( 3-10 年 )

  • TDB + 日銀短観 + 道財務局 BSI の統合データ基盤
  • ifo モデル参照の業種別細分化 ・ 国際比較
  • 中小企業 BCP ・ 投資計画への DI 連動
  • 観光 ・ 一次産業の景気観測指標統合

長期 ( 10 年以上 )

  • 道内独自の長期景気観測指数体系の確立
  • ifo 型業種別 ・ 地域別 ・ 国際比較データ基盤
  • 景気観測を踏まえた長期産業政策設計
  • DI を中心にした道内経済リテラシー教育

6. わたしたちにできること

個人として

  • TDB ・ 日銀短観 ・ 道財務局レポートを継続購読
  • 業種別 DI を投資 ・ キャリア判断に活用
  • 道内経済の月次推移を意識する
  • 地元中小企業の景気感を聞き、 構造的な議論を

企業・組織として

  • DI を経営判断 ・ 投資計画に組み込む
  • 業種別 ・ 地域別の景気構造を BCP に反映
  • 中小企業向け景気指標の透明性向上
  • 従業員の経済リテラシー教育
