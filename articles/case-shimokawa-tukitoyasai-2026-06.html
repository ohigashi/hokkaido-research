下川町 月と野菜 - 元製麺所が「はかり売り」と「移住者起業」の交点に

道内空き家 60 万戸 ・ ゼロカーボン 48% 削減目標 ・ 中山間人口流出の課題に対し、 下川町の月と野菜は元製麺所を活用したオーガニック食材はかり売り + 喫茶で 2024 年 12 月オープン。 移住 10 年の女性店主が示す小規模 ・ 持続型店舗のモデル。

hokkaido-research 編集部

道内は空き家 60 万戸 ・ 中山間商店衰退 ・ ゼロカーボン 48% 削減目標が並走する。 一方で「移住者の起業」を地域に根付かせる仕組みは試行錯誤段階。

上川管内 ・ 下川町の 月と野菜は、 元製麺所の建物を活用した オーガニック食材のはかり売りと喫茶を組合せた小規模店。 2024 年 12 月オープン、 運営者は四国でライフスタイルショップ勤務を経て約 10 年前に下川町へ移住した若園佳子さん。 「お店 = ゼロウェイストの実装」と「移住者起業の地域定着」を同時に体現する取り組み。

1. 道内の課題と数値

道内空き家数は 2023 年住宅 ・ 土地統計で約 60 万戸 ・ 空き家率 17% ( 全国 13.8% )。 中山間 ・ 商店街の空き店舗が継続的な課題。出典: 総務省・住宅 ・ 土地統計調査 ↗
道庁 ゼロカーボン北海道は 2030 年に温室効果ガス 48% 削減 ( 2013 年度比 ) 目標。 ゴミ削減 ・ ゼロウェイスト ・ オーガニック消費が政策課題。出典: 北海道庁・ゼロカーボン北海道 ↗
下川町人口は約 3,000 人。 2017 年 SDGs 未来都市、 2022 年脱炭素先行地域指定。 FSC 森林認証 + バイオマス + 移住促進を 50 年単位で実装。出典: 下川町・公式サイト ↗
仮説: 月と野菜は「空き家活用 + ゼロウェイスト + 移住者起業」の三位一体。 下川町が長年構築してきた SDGs 都市としての文脈基盤が、 小規模店舗の成立を支えている。

2. 月と野菜の構造

2024 年 12 月、 元製麺所の建物を活用してオープン。 商品は約 100 種のオーガニック食材 ( ナッツ ・ ドライフルーツ ・ パスタ等 ) を 20g 単位ではかり売り。 環境配慮型の洗剤 ・ 日用品も。 喫茶スペースで自家製おやつ。出典: 月と野菜 ・ Instagram ↗
店主 若園佳子さんは四国のライフスタイルショップ勤務経験あり、 約 10 年前に下川町へ移住。 「暮らしを大事にしたい」が移住動機。 月 2 回札幌のパン屋から入荷、 道内飲食店コラボイベント ・ 魚屋移動販売も計画。出典: 下川タイムス ・ 月と野菜紹介 ↗

3. なぜ下川町で成立するか

月と野菜単体ではなく、 下川町という基盤との合わせ技で成立している。

  • 下川 SDGs 都市 ・ 脱炭素先行地域の長年の文脈
  • FSC 林業 ・ バイオマス等の環境意識が地域文化に浸透
  • 移住者支援制度 ・ 起業支援の充実
  • 観光 ・ 視察客の継続的な流入 ( 顧客基盤の補完 )
  • 小規模自治体ゆえの直接交渉 ・ 速い意思決定
推測: 月と野菜の成功要因は「店主の経験 + 商品コンセプト + 下川町の文脈基盤」の三位一体。 単独要因では再現困難。 但し、 同様の文脈基盤を持つ自治体 ( 上士幌 ・ 厚真 ・ 美瑛等 ) では参照可能性が高い。

4. 道内への翻訳可能性

  • SDGs ・ 脱炭素先行地域 ( 上士幌 ・ 厚真 ・ 鹿追 ) — 同類型の文脈基盤
  • 観光地 ・ 移住先 ( 美瑛 ・ 富良野 ・ 倶知安 ) — 来訪客基盤あり
  • 中山間商店街 ( 滝上 ・ 中川 ・ 雄武 ) — 空き家活用 + 起業支援
  • 都市近郊 ( 当別 ・ 江別 ・ 北広島 ) — リモートワーク移住者の起業

5. 限界と論点

論点: ( 1 ) 個店ゆえの売上 ・ 経営持続性、 ( 2 ) はかり売り商品の供給チェーン構築、 ( 3 ) 観光客依存リスク、 ( 4 ) 文脈基盤 ( 下川 SDGs 都市 ) なしでの再現難度、 ( 5 ) 移住者起業の世代継承。
