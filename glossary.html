用語集

hokkaido-research で扱う専門用語・概念の解説。記事を読む上で役立つ用語をテーマ別にまとめます。

地域戦略・政策の概念

残る資産 のこるしさん
移住者・取り組みが地域に残す価値を「事業資産・関係資産・仕組み資産・規範資産」等の種類で分類し、定着率より残った資産で評価する考え方。当サイトの編集軸。
関係人口 かんけいじんこう
移住 ( 定住人口 ) でも観光 ( 交流人口 ) でもなく、地域と継続的に関わる人々。ふるさと納税・副業・二地域居住・オンライン参加等、関わり方は多様。
UIJ ターン ゆーあいじぇーたーん
U ターン ( 出身地に戻る ) 、I ターン ( 縁のない地方へ移住 ) 、J ターン ( 出身地近郊に移住 ) の総称。移住政策で使われる。
DMO ディーエムオー
Destination Management Organization。地域観光の戦略策定・マネジメントを行う組織。観光協会の進化形。

財源・制度

ふるさと納税
寄附金税制で、特定自治体に寄附すると所得税・住民税が控除される制度。返礼品が話題になるが、本質は寄附者と自治体の関係構築の仕組み。
GCF ジーシーエフ
ガバメントクラウドファンディング。ふるさと納税を活用した課題解決型寄附。自治体が具体プロジェクト ( 脱炭素・教育・子どもの貧困等 ) を提示し、寄附者が「投資先」感覚で参加。
J-クレジット制度
温室効果ガスの削減・吸収を「クレジット」として認証・取引できる制度。森林吸収量等を経済価値化する仕組み。
脱炭素先行地域
環境省が指定する、2030年までに民生部門の電力消費からの CO2 排出を実質ゼロにする取り組みを行う地域。北海道では上士幌町・ニセコ町等が指定。

企業・組織運営

BCP ビーシーピー
Business Continuity Plan。事業継続計画。災害・事故・パンデミック等の緊急時に事業を継続・早期復旧する計画。
DX ディーエックス
Digital Transformation。デジタル技術を活用した事業・業務・組織の変革。単なる IT 導入ではなく、構造的な変革を指す。
タスクシフト
医療・介護現場で、医師・介護士の業務の一部を看護師・薬剤師等の他職種に分担すること。人材不足対策として注目される。
PPP / PFI ピーピーピー / ピーエフアイ
Public-Private Partnership / Private Finance Initiative。公共サービス・インフラ整備に民間資金・ノウハウを活用する手法。

産業・経済

ラピダス
2022年設立の日本の半導体メーカー。千歳市に 5兆円規模の投資で先端 2nm プロセス半導体製造を目指す。北海道経済への影響大。
サテライトオフィス
企業の本社から離れた地域に設置する小規模オフィス。徳島県神山町・道内の地方都市等への進出が増加。
サーキュラーフード
食品廃棄を出さない循環型食品システム。生産・加工・流通・消費・廃棄 ( 飼料化・堆肥化 ) の全段階を再設計する考え方。
MSC エムエスシー
Marine Stewardship Council。海洋管理協議会。持続可能な漁業に対する国際認証制度を運営。MSC 認証は持続可能な漁業・水産物の証。

地域社会・福祉

地域包括ケアシステム
高齢者が住み慣れた地域で自分らしく暮らし続けられるよう、医療・介護・予防・住まい・生活支援を一体的に提供する仕組み。
こども食堂
子どもに無料・低価格で食事を提供する地域活動。地域住民主導で運営、子どもの貧困・孤食対策・居場所として機能。
ヤングケアラー
本来大人が担う家族の介護・世話を、子ども・若者が日常的に行う状態。学校生活・進路選択への影響が課題。
アイヌ施策推進法
2019年施行。アイヌの人々を「先住民族」と認め、文化・経済・社会的地位の向上を国の責務とする法律。

交通・インフラ

上下分離方式
鉄道で、「線路・駅等のインフラ ( 下 ) 」を公的主体が保有・維持し、「運行 ( 上 ) 」を民間鉄道事業者が担う方式。鉄道存続のモデル。
デマンド交通
利用者の予約に応じて運行する公共交通。バス・タクシーの中間形態。過疎地・高齢化地域の移動手段として注目。
MaaS マース
Mobility as a Service。鉄道・バス・タクシー・シェア自転車等の交通サービスをアプリで統合検索・予約・決済できる仕組み。
コンパクトシティ
居住地・商業・行政機能を中心部に集約し、効率的でコンパクトな都市構造を目指すまちづくり。富山市等が先行。

編集・概念フレーム

中心問いちゅうしんとい
課題の本質を 1つの問いに集約し、議論や施策の判断軸にする編集手法。例: 「移住政策は人を固定することか、人を通じて資産を残すことか」。当サイトの記事・ISSUE で重視。
3段ジャーニーさんだんじゃーにー
「課題を知る → アイデアを得る → 行動に移す」のユーザー体験フレーム。当サイトの設計軸。
残る資産フレームのこるしさんふれーむ
「事業資産・関係資産・仕組み資産・規範資産」の 4分類で施策の長期効果を評価する考え方。

移住・関係人口・暮らし

二地域居住にちいききょじゅう
都市と地方の両方に拠点を持って暮らす働き方・ライフスタイル。関係人口の主要形態の 1つ。
ふるさと回帰ふるさとかいき
都市住民が地方に移住・関わる動き。認定 NPO 法人ふるさと回帰支援センターが推進。
居住誘導区域きょじゅうゆうどうくいき
都市計画法に基づき、人口減少下でも生活サービスや公共交通を維持するため居住を誘導する区域。コンパクトシティの基盤。
移住相談窓口いじゅうそうだんまどぐち
自治体や民間が運営する移住希望者向け相談・マッチング機能。
島留学・地域留学しまりゅうがく・ちいきりゅうがく
島根県海士町等が始めた、地域外の生徒を高校に受け入れる留学プログラム。教育魅力化と移住の入口。

産業・起業・雇用

6次産業化ろくじさんぎょうか
1次 ( 生産 ) + 2次 ( 加工 ) + 3次 ( 販売・サービス ) を統合する地域産業モデル。1 + 2 + 3 = 6。
事業承継じぎょうしょうけい
中小企業・一次産業者の経営を次世代に引き継ぐこと。親族継承・第三者継承等の方法がある。黒字廃業の防止が論点。
VC ( ベンチャーキャピタル )ぶいしー
高成長スタートアップに投資する投資ファンド。地方スタートアップ生態系の重要なプレイヤー。
アクセラレーターあくせられーたー
短期集中型でスタートアップを育成する民間・公的プログラム。資金・メンタリング・ネットワークを提供。
U ターン・I ターン・J ターンゆーたーん・あいたーん・じぇーたーん
U= 出身地に戻る、I= 縁のない地方へ移住、J= 出身地近郊に移住。移住政策で使われる分類。
新規就農しんきしゅうのう
農業を新たに始めること。親元就農・雇用就農・独立就農の 3種類があり、全国新規就農相談センターが支援。

観光・地域経済

オーバーツーリズムおーばーつーりずむ
観光客が過剰に集中することで地域住民の生活・環境・文化に負荷をかける現象。ニセコ・京都等で顕在化。
MICEまいす
Meeting・Incentive travel・Convention・Exhibition の頭文字。ビジネス目的の観光・国際会議の総称。
地域ブランドちいきぶらんど
産品・産地・自治体のブランド化。プレミアム価格化・ロイヤルティ獲得を目指す。
ワーケーションわーけーしょん
Work + Vacation 。リゾート・観光地で仕事を続ける働き方。コロナ後に拡大、地方創生の機会。

教育・子育て

教育魅力化きょういくみりょくか
地域・学校・行政の連携で公立校 ( 特に高校 ) の魅力を高める取り組み。海士町の島前高校が先行事例。
フリースクールふりーすくーる
不登校・学校外学習を支援する民間学習施設。教育機会確保法 ( 2017 ) で位置づけ向上中。
学習支援がくしゅうしえん
経済的困難・発達特性等で学習機会の少ない子ども・若者向けの無料・低価格の学習プログラム。
待機児童たいきじどう
認可保育所の利用希望が満たされない子ども。各市町村が把握・公表する。「潜在待機児童」を含めるとさらに多い。

医療・福祉

無医地区むいちく
医療機関のない山間部・過疎地。北海道に約 137ヶ所存在。
8050 問題はちまるごーまるもんだい
80 代の親と 50 代のひきこもりの子の同居・親亡き後の生活困窮リスク。
認知症サポーターにんちしょうさぽーたー
認知症の人・家族を地域で見守るためのボランティア。養成講座を受講して取得。
在宅医療・訪問介護ざいたくいりょう・ほうもんかいご
病院・施設ではなく、患者の自宅で行う医療・介護。地域包括ケアの中核。

環境・エネルギー

再生可能エネルギーさいせいかのうえねるぎー
風力・太陽光・バイオマス・地熱等、自然由来で枯渇しないエネルギー。北海道はポテンシャル全国一。
系統制約けいとうせいやく
発電された電力が送電網の容量不足で送れない状況。北海道では本州との連系線が細い問題。
サーキュラーエコノミーさーきゅらーえこのみー
廃棄を出さず、資源を循環させる経済モデル。脱炭素・資源効率化と並ぶ国際潮流。
ブラックアウトぶらっくあうと
電力系統全体の崩壊による広域停電。2018年 9月の北海道胆振東部地震で全道停電を経験。

デジタル・サイバー

ランサムウェアらんさむうぇあ
システムを暗号化し身代金を要求するサイバー攻撃。地方自治体・中小企業・医療機関を狙う事例増加。
SOC ( セキュリティオペレーションセンター )えすおーしー
サイバー攻撃を 24時間監視・検知・対応する専門組織。共同 SOC が地方の有効な対策。
ゼロトラストぜろとらすと
「すべての通信・ユーザーを信頼しない」前提のセキュリティモデル。クラウド・リモート時代の主流。
スマートシティすまーとしてぃ
IoT・AI・ビッグデータを活用した都市運営。脱炭素・効率・暮らしやすさを統合的に向上。
オープンデータおーぷんでーた
公開された行政・公共データ。誰でも利用・加工・再配布可能。シビックテック・地域 DX の基盤。

災害・防災

レジリエンスれじりえんす
災害・ショックからの回復力・適応力。インフラ多重化・コミュニティ・BCP 等の総合的な能力。
自助・共助・公助じじょ・きょうじょ・こうじょ
防災・福祉の 3段階。自助 ( 自分 )・共助 ( 地域 )・公助 ( 行政 ) の役割分担。
避難計画ひなんけいかく
災害時の避難経路・場所・物資等の事前計画。自治体策定だが家庭・企業も独自の計画が必要。

アイヌ・多文化

ウポポイうぽぽい
国立アイヌ民族博物館 + 国立民族共生公園。白老町に 2020年開設、アイヌ文化発信の中核。
ユニバーサルデザインゆにばーさるでざいん
障害・年齢・性別・言語に関わらず利用できる商品・サービス・環境の設計。インクルーシブの基盤。
やさしい日本語やさしいにほんご
簡単な語彙・文法で書く日本語。外国人住民・障害者等の情報アクセスを支える。
